# 🍲 Recipe API

A RESTful Web API built with ASP.NET Core and Entity Framework Core.

## Features

* Get all recipes
* Get recipe by id
* Create recipe
* Update recipe
* Delete recipe
* SQL Server database
* Entity Framework Core
* Swagger documentation
* CORS enabled

## Technologies

* ASP.NET Core 8
* Entity Framework Core
* SQL Server
* Swagger/OpenAPI

## Endpoints

| Method | Endpoint         | Description      |
| ------ | ---------------- | ---------------- |
| GET    | /api/Recipe      | Get all recipes  |
| GET    | /api/Recipe/{id} | Get recipe by id |
| POST   | /api/Recipe      | Create recipe    |
| PUT    | /api/Recipe/{id} | Update recipe    |
| DELETE | /api/Recipe/{id} | Delete recipe    |
